# Prompts para Modos de Juego - Tap Frenzy

Este documento contiene los prompts de diseño para los tres modos de juego del proyecto Tap Frenzy.

---

## Modo 1: Clásico 🍺

### Descripción del Modo
El modo original del juego. Sin límite de tiempo, con zona de peligro activada. Ideal para jugadores que quieren una experiencia tradicional de fusión de botellas.

### Prompt de Diseño

```
Crea una imagen de fondo para el modo de juego "Clásico" de un juego de fusión de botellas de cerveza artesanal.

ESTILO VISUAL:
- Estilo: Retro arcade meets cervecería artesanal
- Paleta: Tonos ámbar, dorados y marrones cálidos
- Atmósfera: Acogedora, tradicional, con vibración de bar clásico
- Textura: Madera envejecida, cobre brillante, espuma de cerveza

ELEMENTOS CLAVE:
- Fondo de tablones de madera de barril de cerveza
- Sutil brillo dorado que evoca la cerveza recién servida
- Siluetas de botellas de cerveza de diferentes tamaños en el horizonte
- Partículas de luz flotantes como burbujas de cerveza
- Bordes con efecto de viñeta oscura para enfoque central

MOOD:
- Nostálgico y acogedor
- Como entrar a tu cervecería favorita
- Tradición y maestría cervecera

ESPECIFICACIONES TÉCNICAS:
- Resolución: 1920x1080 ( adaptable a 16:9 )
- Formato: PNG con transparencia opcional para overlay
- Optimizado para web (menos de 500KB)
```

### Prompt para Icono/Logo del Modo

```
Diseña un icono circular para el modo "Clásico" de un juego móvil.

ELEMENTOS:
- Jarra de cerveza espumosa en el centro
- Estilo cartoon con contorno grueso negro
- Fondo circular con degradado ámbar-dorado
- Pequeñas burbujas flotando alrededor
- Brillo sutil en la espuma

COLORES:
- Principal: #D4A017 (ámbar cerveza)
- Secundario: #F5E6C8 (espuma)
- Contorno: #2C1810 (marrón oscuro)
- Acento: #FFD700 (dorado brillante)

ESTILO: Vector plano, friendly, reconocible a 64x64px
```

## Modo 2: Rápido ⚡

### Descripción del Modo
Modo contrarreloj de 90 segundos. Cada fusión añade +5 segundos, y fusiones de tier 6 o superior otorgan +10 segundos. Con zona de peligro activada. Para jugadores que buscan adrenalina.

### Prompt de Diseño

```
Crea una imagen de fondo para el modo de juego "Rápido" de un juego de fusión de botellas de cerveza.

ESTILO VISUAL:
- Estilo: Cyberpunk meets deportes electrónicos
- Paleta: Neón azul eléctrico, morado, naranja intenso
- Atmósfera: Adrenalina pura, velocidad, competición extrema
- Efectos: Rayos de velocidad, estelas de luz, motion blur sutil

ELEMENTOS CLAVE:
- Líneas de velocidad diagonales que cruzan la pantalla
- Efectos de partículas de energía eléctrica
- Reloj digital estilizado fragmentado en el fondo
- Circuitos y líneas de neón que pulsan
- Explosiones de luz naranja y azul en los bordes
- Efecto de "warp speed" o hipervelocidad sutil

MOOD:
- Intenso y competitivo
- "Corre contra el tiempo"
- Estilo arcade de los 80s modernizado
- E-sports aesthetic

ESPECIFICACIONES TÉCNICAS:
- Resolución: 1920x1080 ( adaptable a 16:9 )
- Formato: PNG con transparencia opcional para overlay
- Optimizado para web (menos de 500KB)
- Debe funcionar bien con efectos de animación rápidos encima
```

### Prompt para Icono/Logo del Modo

```
Diseña un icono circular para el modo "Rápido" de un juego móvil.

ELEMENTOS:
- Rayo eléctrico estilizado en el centro
- Cronómetro o reloj de arena detrás del rayo
- Estilo cartoon dinámico con efecto de movimiento
- Líneas de velocidad saliendo del centro
- Brillo eléctrico en los bordes

COLORES:
- Principal: #00D4FF (cian eléctrico)
- Secundario: #FF6B35 (naranja adrenalina)
- Fondo: #1A0A2E (morado oscuro profundo)
- Contorno: #FFFFFF (blanco brillante)
- Acento: #FFD700 (destello dorado)

ESTILO: Vector plano con efectos de brillo, dinámico, legible a 64x64px
```

## Modo 3: Zen 🧘

### Descripción del Modo
Modo relajado sin zona de peligro y sin límite de tiempo. Ideal para practicar, desconectar o jugar sin presión. La experiencia mindfulness de Tap Frenzy.

### Prompt de Diseño

```
Crea una imagen de fondo para el modo de juego "Zen" de un juego de fusión de botellas de cerveza.

ESTILO VISUAL:
- Estilo: Minimalista japonés meets naturaleza tropical
- Paleta: Verdes suaves, turquesas, arena clara, blanco roto
- Atmósfera: Tranquilidad absoluta, meditación, flujo natural
- Textura: Acuarela suave, papel arroz, bambú

ELEMENTOS CLAVE:
- Fondo de jardín zen con arena rastrillada en patrones ondulantes
- Bambú y hojas tropicales suaves en los bordes
- Reflejo de agua tranquila con pequeñas ondas concéntricas
- Pétalos de flor cayendo suavemente (ceremonia del té)
- Montañas suaves en la distancia estilo ukiyo-e
- Espacio negativo abundante, nada sobrecarga visual

MOOD:
- Totalmente relajado y meditativo
- "Respira y disfruta"
- Conexión con la naturaleza
- Mindfulness y presencia

ESPECIFICACIONES TÉCNICAS:
- Resolución: 1920x1080 ( adaptable a 16:9 )
- Formato: PNG con transparencia opcional para overlay
- Optimizado para web (menos de 500KB)
- Colores muy suaves que no cansen la vista
```

### Prompt para Icono/Logo del Modo

```
Diseña un icono circular para el modo "Zen" de un juego móvil.

ELEMENTOS:
- Figura meditando en posición loto (silueta simple)
- Círculo de iluminación/aura alrededor
- Hoja de bambú o loto como elemento decorativo
- Ondas concéntricas estilo piedra en estanque
- Estilo minimalista y sereno

COLORES:
- Principal: #7CB342 (verde bambú suave)
- Secundario: #4DB6AC (turquesa tranquilo)
- Fondo: #E8F5E9 (verde muy pálido)
- Contorno: #2E7D32 (verde bosque)
- Acento: #FFF8E1 (luz cálida dorada suave)

ESTILO: Vector plano minimalista, orgánico, calmado, claro a 64x64px
```

## Notas de Implementación

### Archivos Necesarios

Para cada modo se necesitan:
1. **Fondo principal** (1920x1080) - Usar en Game.tsx como background
2. **Icono del modo** (64x64, 128x128, 256x256) - Usar en GameModeSelector.tsx
3. **Miniatura** (para selector, 200x150 aprox)

### Rutas Sugeridas

```
public/
  assets/
    modes/
      classic/
        background.png
        icon.png
        thumbnail.png
      quick/
        background.png
        icon.png
        thumbnail.png
      zen/
        background.png
        icon.png
        thumbnail.png
```

### Integración en Código

Los prompts están diseñados para generar assets que encajen con:
- `GameModeConfig` en `src/types/gameModes.ts`
- `GameModeSelector` en `src/components/GameModeSelector.tsx`
- Sistema de fondos dinámicos en `Game.tsx`

### Herramientas Recomendadas

- **Midjourney**: Excelente para fondos atmosféricos
- **DALL-E 3**: Bueno para iconos y elementos específicos
- **Stable Diffusion**: Para iteraciones y ajustes finos
- **Figma/Illustrator**: Para vectorizar iconos si es necesario
