# Prompts para Imágenes de Poderes

Este documento contiene prompts detallados para generar imágenes de los poderes del juego.

---

## 🔄 Power: Swap (Intercambio)

### Prompt en Inglés (Recomendado para IA)
```
Game ability icon for "Swap" power, two beer bottles rotating in a circular swap motion with arrows, orange and amber color scheme, magical particle effects, hexagonal frame with beer foam border, mobile game UI style, transparent background, high detail, stylized illustration, game asset
```

### Descripción
Icono que representa el poder de intercambiar la botella actual. Debe mostrar dos botellas de cerveza en movimiento circular con flechas indicando el intercambio, con efectos de partículas mágicas y un marco hexagonal con bordes de espuma.

### Especificaciones
- **Tamaño sugerido:** 128x128px o 256x256px
- **Estilo:** Icono de UI para móvil
- **Colores:** Naranja, ámbar, dorado
- **Fondo:** Transparente (PNG)

## 💣 Power: Bomb (Bomba)

### Prompt en Inglés (Recomendado para IA)
```
Game ability icon for "Bomb" power, explosive beer barrel with burning fuse, dynamic explosion particles and fire sparks, red and orange color scheme with black accents, warning symbols and blast radius indicators, hexagonal frame with scorched metallic border, mobile game UI style, transparent background, high detail, stylized illustration, game asset
```

### Especificaciones
- **Tamaño sugerido:** 128x128px o 256x256px
- **Estilo:** Icono de UI para móvil
- **Colores:** Rojo, naranja, negro, amarillo fuego
- **Fondo:** Transparente (PNG)

## Notas Generales

### Herramientas Recomendadas
- **Midjourney:** Excelente para estilos de juego y consistencia
- **DALL-E 3:** Buen control de los elementos solicitados
- **Stable Diffusion:** Personalización total con LoRAs de game assets
- **Adobe Firefly:** Buena integración con flujos de trabajo profesionales

### Parámetros Sugeridos (Midjourney)
```
--ar 1:1 --v 6 --style raw --s 250
```

### Formato de Salida
- **Formato:** PNG con transparencia
- **Resolución:** 512x512px mínimo (luego escalar a 128x128px para el juego)
- **Estilo visual:** Consistente con la temática cervecera del juego

---

## Referencia de Estilo del Juego

Los iconos deben mantener coherencia con:
- Paleta cálida (naranjas, amarillos, marrones)
- Bordes de espuma/beer
- Temática de bar/cervecería
- Estilo cartoon/stylized (no realista)
- Marco hexagonal para consistencia visual
